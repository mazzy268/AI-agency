# AI Agency Command Center / AgencyOS

This repository is the starting point for building **AgencyOS**: a premium AI agency operating system for selling, building, and delivering automation systems to UK small businesses.

## Goal

Build a web dashboard that helps an AI automation agency:

- Capture and manage local business leads
- Diagnose business problems
- Recommend profitable automation workflows
- Calculate ROI
- Generate client proposals
- Create implementation blueprints
- Track client pipeline and delivery progress

## Target niche

Initial focus: UK trades and local service businesses, including plumbers, electricians, roofers, landscapers, builders, cleaners, dentists, salons, gyms, estate agents, and other appointment/lead-driven SMBs.

## Core MVP flow

1. Add a business lead
2. Diagnose their missed revenue/problem areas
3. Recommend the best automation to sell them
4. Estimate monthly ROI
5. Generate a client-facing proposal
6. Generate an internal implementation checklist
7. Track pipeline and delivery status

## Priority automation offers

- Missed call recovery system
- Quote follow-up automation
- Review generation automation
- Appointment reminder system
- Lead capture and CRM sync
- Facebook lead ad follow-up
- Website chatbot to booking flow
- Invoice/payment chase automation
- Old customer reactivation campaign

## Build principles

- £0 capital MVP
- No paid APIs required at the start
- Mock/local data first, Supabase-ready later
- Practical ROI-focused features only
- Premium, modern, mobile-responsive dashboard UI
- Clean TypeScript and reliable app structure
